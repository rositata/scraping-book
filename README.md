# scraping-books
Praktikum **Web Scraping** untuk mengambil data dari website [Books to Scrape](https://books.toscrape.com/).

## Tujuan
Tugas ini merupakan bagian dari praktikum mata kuliah Prediksi Modern and Marchine Learning menggunakan Python. 
Program ini dirancang untuk mengambil data sebanyak **1000 buku** dengan variabel (judul, harga, stok, rating, dan URL) dan menyimpannya dalam bentuk DataFrame. dari situs *Books to Scrape* dan menyimpannya dalam bentuk **DataFrame** serta file **Excel (.CSV)**.
